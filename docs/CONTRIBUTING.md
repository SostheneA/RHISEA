## How to Contribute

### Report Bugs

- Open a [new issue](https://github.com/SostheneA/RHISEA/issues/new)
- Include: R / RHISEA version, reproducible code, `baseline` / `mixture`
  data, full error message

### Development Standards

- Requires R (\>= 3.5.0)

### Installation

``` r
devtools::install_github("SostheneA/RHISEA")
```
